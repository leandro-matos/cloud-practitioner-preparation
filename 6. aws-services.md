
* Mais de 25 Produtos Disponíveis, os principais que caem na prova:

- Governança;
- Gerenciamento de Custos AWS;
- Integração de Aplicativos;
- Segurança e Conformidade;

=====================================================================================================================

- Computação e Rede:
    1) Amazon Elastic Cloud Compute (EC2):
        - Serviço da Web que fornece capacidade computacional redimensionável;
    2) AWS Lambda:
        - Plataforma de back-end sem gerenciamento de servidores para execução de códigos;
    3) AWS Elastic Load Balacing (ELB):
        - Distribui automaticamente o trafego de aplicativos recebidos de várias instâncias;
    4) AWS Elastic Beanstalk:
        - Plataforma para disponibilizar seu código/aplicativo na Web da forma mais simples possível;
    5) AWS Virtual Private Network (VPC):
        - Definição de redes isoladas para uma maior segurança na sua infraestrutura;
    6) AWS Direct Connect:
        - Permite que organizações estabeleçam comunicação de seus datacenter com a AWS;
    7) AWS Route53:
        - Servço Web de DNS (Domain Name System), também permite que compre e gerencie domínios;

- Armazenamento e Entrega de Conteúdo;
    8) AWs Simple Storage Service (S3):
        - Fornece armazenamento de objetos altamente durável e escalável (armanzenas objetos, sites http, etc);
    9) AWS Glacier:
        - Armazenamento seguro para arquivamento de dados e backup a longo prazo;
    10) Amazon Elastic Block Store (EBS):
        - Volumes persistentes de armazenamento em nível de bloco para uso de instâncias ec2;
    11) AWS Storage Gateway:
        - Serviço que conecta um dispositivo de software local com armazenamento baseado em nuvem. Integrando com amazon s3;
    12) Amazon CloudFront:
        - Serviço de Web para entrega de conteúdo com baixa latência, altas transferências de dados por meio de Edge Locations;

- Banco de Dados:
    13) Amazon Relational Database (RDS):
        - Fornecimento de um banco de dados relacional totalmente gerenciado, o usuário não se preocupa com tarefas operacionais;
    14) Amazon DynamoDB:
        - Serviço de base de dados nosql com suporte a modelos de chave/valor;
    15) Amazon Redshift:
        - Serviço de DataWarehouse rápido e em escala de petabytes;
    16) Amazon ElastiCache:
        - Armazenamento de dados na memória totalmente gerenciado, compatível com Redis ou Memcached.

- Gerenciamento:
    17) Amazon CloudWatch:
        - Serviço de monitoramento dos recursos da AWS e dos aplicativos;    
    18) AWS CloudFormation:
        - Serviço para infra as code (Provisionar e Gerenciar a infra de forma ordenada);
    19) AWS CloudTrail:
        - Serviço Web que registra as chamadas na API da AWS, utilizado para auditoria;
    20) AWS Config:
        - Serviço que oferece inventário de recursos, histórico de configurações e notificações de alterações;

- Segurança e Identidade;
    29) AWS Identity and Access Management (IAM):
        - Controlar com seguraça o acesso aos serviços da AWS com usuários, grupos, profiles;
    30) AWS Key Management Service (KMS):
        - Serviço gerenciado para criar e controlar as chaves de criptografia para criptografas os dados;
    31) AWS Directory Service (AWS):
        - Permitir que seja configurado o AD na nuvem ou se conectar a um local já existente;
    32) AWS Certificate Manager:
        - Serviço que permite que as organizações gerenciem certificados TLS/SSL para uso de serviços;
    33) Aws Web Application Firewal (WAF):
        - Ajuda a proteger aplicações Web contra ataques, permite bloquear tráfego e definir regras customizáveis de acesso;

- Serviços de Aplicação:
    34) Amazon API Gateway:
        - Serviço para gerenciar, criar, publicar e manter API's em qualquer escala;
    35) Amazon Elastic Transcoder:
        - É a transcodificação de mídia na nuvem. É possível "transcodificar" arquivos de mídia de seu original para outros formatos.
    36) Amazon Simple Notification Service (SNS):
        - Serviço Web que controla e gerencia a entrega ou envio de mensagens aos destinatários;
    37) Amazon Simple Email Service (SES):
        - Serviço de e-mail para mensagens transacionais, marketing, etc. Pode ser usado em conjunto com o S3 e SNS;
    38) Amazon Simple Workflow Service (SWF):
        - Criar, executar e dimensionar tarefas em segundo plano com etapas paralelas/sequenciais;
    39) Amazon Simple Queue Service (SQS):
        - Serviço de enfileiramento de mensagens, ajudando no desacoplamento de aplicativos na nuvem.
